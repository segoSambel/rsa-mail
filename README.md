# RSA Mail

## Description
Project UAS PABW Univesitas Muhammmadiyah Sidoarjo.


## Usage

1. Clone this repository
```bash
git clone https://github.com/segoSambel/rsa-mail.git
```

2. Install dependencies
```bash
npm install
```

3. Create `.env` file based on `.env.example`


4. Run the app
```bash
npm start
```
